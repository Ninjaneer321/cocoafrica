African Masks
The African Masks dataset is curated as a set of images depicting historic mask desings from various parts of Africa. It is intended as an interesting addition to datasets that explore the intersection of Art and AI.

Similarity is measured by comparing the cosine distance of features extracted (using a VGG network ) from the generated image and images in the dataset.
