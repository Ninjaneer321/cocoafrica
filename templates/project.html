Project Description
This work presents the COCOA dataset created with the goal of improving computer vision tasks ( detection, captioning, classification etc) with a focus on the African context. Following examples from existing datasets, this work focuses on gathering images of complex scene and containing objects that are more likely to be observed within Africa and largely underrepresented in other repositories. The COCOA-Africa dataset can be used to augment existing datasets to improve representativeness for general classification and object detection tasks. In addition, the dataset allows a focus on the classification and detection categories that are meaningful for the Africa context.
